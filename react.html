課程主題與大綱

隨著網頁技術的提升，網頁前端的功能需求隨之提升，複雜度也隨之提高。
為此，許許多多的前端套件為此而生，較有名的像是 angular , vue , react 等等。
而這次要講的就是 react 這套 framework, 利用簡單好懂的 flux 架構, 了解 react 的運作原理並且實作一款應用。
